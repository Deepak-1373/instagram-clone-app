- Firebase

## Getting Started

- Clone the repository on your local machine with the command below in your terminal, and cd into the **instagram-clone** folder

```sh
https://github.com/Deepak-1373/instagram-clone-app.git

cd instagram-clone
```

- Install dependencies

```sh
npm install
```

- Start the Server🚀

```
npm start
```
